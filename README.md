# Playwright tests

Run from the **home directory** (recommended):

```bash
cd ~
npm test
# or
npx playwright test
```

Run from this folder:

```bash
cd ~/playwright-tests
npm test
```

**pogosobackend** API tests live at [`../pogosobackend/e2e`](../pogosobackend/e2e):

```bash
npm run test:pogoso
```
